About STEPBible Explorer

This application provides access to the STEPBible data, which includes lexicons, tagged texts, morphology codes, and proper names information for biblical study.

The data is derived from the STEPBible project, which provides high-quality, openly licensed biblical resources for students, teachers, and pastors.

Learn more about the STEPBible project at stepbible.org.
